# C-Straboscope

# -> Страбоскоп с помощью светодиодов и потенциометра.
# 
# -> Компоненты для сборки проекта:
# - Макетная плата
# - Arduino UNO
# - Светодиоды (3шт красные, 3шт зеленые)
# - Потенциометр 50кОм
# - Перемычки
# - Резисторы 220Ом

# -> Логика работы
# Для подачи сигнала на светодиоды используется 2 пина - 10 и 11.
# Пины подключены к макетной плате. На макетной плате последовательно соединены светодиоды.
# В коде прописана логика - сначала два раза мигает один цвет, затем другой. 
# Между подачами команд на пины есть delay - остановка работы программы. В данный метод передается значение с потенциометра, преобразованное в числа от 10 - 500.
# В зависимости от задержки реализуется мигание светодиодов.
#
# -> Схема так же построена на Tinkercad - https://www.tinkercad.com/things/3zjlXR6uzau-frantic-turing-jarv
